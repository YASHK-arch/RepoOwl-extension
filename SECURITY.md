# Security Policy

## Supported Versions

Please use the latest version of the extension to ensure you receive the most recent security updates. If you are using an older version, please upgrade before reporting a vulnerability.

## How to Report Vulnerabilities

If you discover a security vulnerability within RepoOwl, please do not disclose it publicly.

Instead, please send an email to the project maintainers directly or use the GitHub Security Advisory feature in this repository.

## Disclosure Policy

We will attempt to acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. 
Once a fix has been developed and tested, we will publish a security advisory and notify users to update.

## Expected Response Time

- Acknowledgment: 48 hours
- Initial Assessment: 1 week
- Fix Delivery: Dependent on severity (usually 1-3 weeks)
